# VARIÁVEIS:
	* Var => Modo antigo de declarar variáveis (NÃO USAR POIS PODE CAUSAR ALGUNS ERROS);
	* Let => Modo moderno de declarar variáveis;
	* Const => Variáveis com dados que não podem ser alterados.

	OBS - [VARIÁVEIS DECLARADAS DENTRO DE UMA FUNÇÃO SÓ EXISTEM DENTRO DELA]

# FUNÇÕES:
	* Básico:
		function nome_função{
			return x;
		}
		=> Com ou sem parâmetro;
		=> Com ou sem retorno;

	* Arrow Functions:
		1. (a, b) => a * b; - [UMA LINHA DE FUNÇÃO];
		2. (a, b) => {
			[MAIS LINHAS DE FUNÇÃO]
		};
